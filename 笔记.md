## 常用命令
+ npm init
    + npm init-y可以跳过向导，快速生成
+ npm install 
    + 一次性把dependencies选项中的依赖全部安装
+ npm install 包名
    + 只下载
+ npm install --save    `i=install`
    + 下载并保存依赖项
+ npm uninstall 包名    `un=uninstall`
    + 只删除包名 ，其他同上
